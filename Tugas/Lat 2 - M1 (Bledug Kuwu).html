Api Abadi Mrapen

Api Abadi Mrapen merupakan sebuah kompleks yang terletak di Desa Manggarmas, Kecamatan Godong, Kabupaten Grobogan. Kompleks Api Abadi Mrapen merupakan fenomena geologi alam berupa keluarnya gas alam dari dalam tanah yang tersulut api sehingga menciptakan api yang tidak pernah padam walaupun turun hujan sekalipun. Banyak peristiwa besar yang mengambil api dari kompleks ini sebagai sumber obornya, misalnya PON X tahun 1981, PON XIV tahun 1996, dan Asian Games 2018.
